# Agente Virtual Maternidad

Proyecto web interno para acompañar a las personas trabajadoras de Plain Concepts en los pasos principales relacionados con maternidad, paternidad, nacimiento y cuidado del menor.

## Archivos incluidos

- `index.html`: estructura principal de la web.
- `style.css`: estilos visuales.
- `script.js`: funciones interactivas.
- `logo-plain-concepts.png`: logo de Plain Concepts, si está incluido.

## Cómo publicarlo en GitHub Pages

1. Crea un repositorio llamado `agente-virtual-maternidad`.
2. Sube todos los archivos del proyecto.
3. Ve a **Settings > Pages**.
4. En **Source**, selecciona **Deploy from a branch**.
5. En **Branch**, selecciona `main` y `/root`.
6. Pulsa **Save**.

La URL será similar a:

`https://TU-USUARIO.github.io/agente-virtual-maternidad/`

- Explicación de las 19 semanas del permiso.
